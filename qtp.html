Here are some useful links for QTP scripting.

DotNetFactory Objects

DotNetFactory is a new utility object introduced in QTP 9.2 and available in later versions, which enables QTP scripting to directly access methods and properties of a .NET object by creating an instance of this object. You will be amazed using this as you won't realize how easy it could be.

You can create your own custom forms for script execution

  • http://www.learnqtp.com/dotnetfactory-qtp-part1-basics/

Excel Macros

You can use excel macro to make your work easier in QTP script execution

  • https://munibala.wordpress.com/tag/run-macro-using-qtp/

Sikuli

Sikuli automates anything you see on the screen. It uses image recognition to identify and control GUI components. It is useful when there is no easy access to a GUI's internal or source code.

You can use Sikuli to automate your operations.

  • http://www.sikuli.org/
